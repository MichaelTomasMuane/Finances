# Sobre o projecto template - Tema Finances BLOGGER

O **Finances** é um tema profissional desenvolvido especificamente para empresas de contabilidade, consultoria financeira e auditoria que desejam estabelecer uma presença digital moderna e credível. O tema foi projetado para transmitir confiança, profissionalismo e expertise no setor financeiro.. 

Esse tema resolve o problema de baixa credibilidade e excesso de animações em outros temas desfocando o foco do usuario no website,    . 

Este tema é ideal para Pequenos negócios, Escritórios de contabilidade, Consultores financeiros independentes, Empresas de auditoria, Profissionais de gestão financeira e PMEs que oferecem serviços financeiros de subscrição. 

## Estrutura do Projeto

### Tipo de Projeto
Tema XML completo para Google Blogger

Ficheiro único: `Finances.xml`

### Organização Interna
Utilização extensiva de: `<b:section>` `<b:widget>` `<b:include>` `<b:if>`

Separação lógica entre: `Estrutura HTML` `Estilos CSS` `Scripts JavaScript` `Lógicas Condicionais` 

Fácil manuntabilidade: `Código comentado` `Estrutura modular` `cores e fontes editaveis pelo personalizador Blogger`

Tecnologias: `Google Blogger` `XML` `HTML5` `CSS3` `JavaScript (Vanilla)`

## Como instalar

### Pré-requisitos 
tenha uma conta activa do google, entre e criei um novo blog no google blogger (Tenha acesso ao paínel e uma conta administrativa).

Passo 1: Baixar o Tema

Aceda ao repositório do projeto
Faça download do ficheiro Finances.xml
Guarde numa pasta acessível no seu computador

Passo 2: Importar no Blogger

Aceda ao Blogger.com
Selecione o seu blog
No menu lateral, clique em Tema
Clique no botão ⋮ (três pontos) ao lado de "Personalizar"
Selecione Restaurar > Carregar
Escolha o ficheiro Finances.xml
Clique em Carregar
